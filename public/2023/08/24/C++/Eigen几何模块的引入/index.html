数学理论上所出来的angle_vector对应的矩阵应该为：

[0 -1 0]

[1 0 0]

[0 0 1]

但程序输出为：

6.12323e-17 -1 0

1 6.12323e-17 0

0 0 1

将程序中精度由d改成f后，程序输出为：

-4.37114e-08 -1 0

1 -4.37114e-08 0

0 0 1

其中，本应该矩阵中应该为0的元素出现了一个极小值，而且该极小值会随着精度的变化而变化，不过目前发现的影响只有对0元素的影响，这个极小值可以看成0.
